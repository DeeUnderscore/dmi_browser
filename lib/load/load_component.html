You can either load a file from disk, or fetch a file from a remote URL. All fetching is done clientside.

Please note that downloading from remote hosts will either fail or succeed, depending on the CORS configuration of the remote host. If you paste a github.com URL (such as from a "Download" button), the tool will attempt to guess the corresponding raw.githubusercontent.com URL and download from that.
